## Hi there 👋

We are on a journey to make great adventures with great games.

Our current main project is [Stonebreak](https://github.com/Sonar-Arts/Stonebreak), a Java-based 3D voxel-based sandbox RPG inspired by the games Minecraft, Neverwinter Nights, and Dwarf Fortress. We envisioned this game with the idea of creating an easily moddable sandbox RPG, with a living world that grows with or without player input. 
It is a tall task, but we are in it for the long haul. We hope that any and all who are interested will join us on this epic journey. 

We also had another project going called [Ping](https://github.com/Sonar-Arts/Ping) that we were initally working on before swichting to Stonebreak. That project is still being worked on, but on a more sporadic manner as it is being converted away from Python to C++.
